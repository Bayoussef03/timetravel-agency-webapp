# TimeTravel Agency - Webapp Interactive

Webapp interactive pour une agence de voyage temporel fictive, réalisée dans le cadre d’un projet pédagogique de M2.
Le site permet de découvrir plusieurs destinations temporelles, d’interagir avec un assistant conversationnel et d’obtenir une recommandation personnalisée grâce à un quiz interactif.

## Description du projet

L’objectif du projet est de concevoir une webapp moderne et immersive pour l’agence fictive TimeTravel Agency.
L’application met en avant plusieurs destinations temporelles, une interface orientée expérience utilisateur, un chatbot IA et une fonctionnalité de personnalisation.

## Technologies utilisées

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel pour le déploiement
- Mistral AI API pour la partie conversationnelle

## Fonctionnalités implémentées

- Landing page interactive
- Présentation de l’agence
- Galerie de destinations temporelles
- Cartes de destinations
- Chatbot conversationnel
- Quiz de recommandation personnalisée
- Section de réservation

## Outils IA utilisés

- v0.dev : génération initiale de composants et de structure d’interface
- Mistral AI API : intégration du chatbot conversationnel
- IA générative : assistance au développement, à la rédaction de prompts et à l’amélioration du code

## Installation

1. Cloner le repository
2. Installer les dépendances avec la commande :
   ```bash
   npm install
   ```
3. Lancer le serveur de développement :
   ```bash
   npm run dev
   ```
4. Ouvrir l’application dans le navigateur à l’adresse :
   `http://localhost:3000`

## Déploiement

Le projet est déployé en ligne via Vercel.
